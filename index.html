Modul cepat memahami semua jenis function JavaScript untuk ujian.

1. Function Declaration

Paling basic

Penjelasan:
Function declaration adalah function biasa yang dibuat dengan keyword function dan memiliki nama. Ini jenis function paling dasar di JavaScript dan paling sering dipakai di soal ujian.

Ciri-ciri:

  • Menggunakan kata function
  • Memiliki nama function
  • Bisa dipanggil berkali-kali
  • Bisa dipanggil sebelum ditulis
function halo(){
  console.log("Hello");
}

2. Arrow Function

Versi modern

Penjelasan:
Arrow function adalah versi modern dari function yang lebih singkat dan cepat ditulis. Biasanya digunakan di JavaScript modern.

Ciri-ciri:

  • Menggunakan tanda =>
  • Disimpan di variabel
  • Penulisan lebih pendek
  • Sering dipakai di coding modern
const halo = () => {
 console.log("Hello");
}

3. Function Expression

Function dalam variabel

Penjelasan:
Function expression adalah function yang disimpan ke dalam variabel. Mirip arrow function tapi menggunakan keyword function.

Ciri-ciri:

  • Disimpan dalam variabel
  • Menggunakan keyword function
  • Tidak bisa dipanggil sebelum dibuat
const tes = function(){
 console.log("Hai");
}

4. Anonymous Function

Tanpa nama

Penjelasan:
Anonymous function adalah function tanpa nama. Biasanya digunakan sekali pakai untuk event atau proses cepat.

Ciri-ciri:

  • Tidak memiliki nama
  • Digunakan langsung
  • Sering dipakai di tombol & event
button.onclick = function(){
 alert("Klik");
}

5. Callback Function

Function di dalam function

Penjelasan:
Callback adalah function yang dikirim sebagai parameter ke function lain, lalu dijalankan di dalamnya.

Ciri-ciri:

  • Ada function di parameter
  • Dipanggil oleh function lain
  • Sering dipakai untuk validasi & proses
function proses(a,b,cb){
 cb(a,b);
}

6. IIFE

Langsung jalan

Penjelasan:
IIFE (Immediately Invoked Function Expression) adalah function yang langsung berjalan saat dibuat tanpa dipanggil.

Ciri-ciri:

  • Ada tanda ( ) di akhir
  • Langsung berjalan
  • Tidak perlu dipanggil
(function(){
 console.log("Auto run");
})();

7. Function Constructor

Buat object

Penjelasan:
Digunakan untuk membuat object seperti siswa, produk, user. Seperti blueprint.

Ciri-ciri:

  • Menggunakan this
  • Dipanggil dengan new
  • Biasanya huruf awal besar
function Siswa(nama){
 this.nama = nama;
}
let s = new Siswa("Budi");

8. Async Function

Delay / tunggu proses

Penjelasan:
Async function digunakan untuk proses yang membutuhkan waktu seperti loading data atau delay.

Ciri-ciri:

  • Menggunakan async
  • Menggunakan await
  • Bisa delay proses
async function test(){
 await new Promise(r=>setTimeout(r,1000));
 console.log("Selesai");
}
